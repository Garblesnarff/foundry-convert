# foundry-convert: Design System Reference

See `~/Projects/foundry/_shared/DESIGN_SYSTEM.md` for full Foundry design system.

## Specific Overrides for foundry-convert

### Color Usage
- **Primary**: Charcoal (#141210)
- **Accent**: Forge Amber (#E8A849)
- **Success**: Green (#4CAF50) — conversion complete
- **Error**: Red (#DC3545) — conversion failed

### Typography
- **Headlines**: Playfair Display, 24px, 700 weight
- **Labels**: DM Sans, 14px, 400 weight
- **Technical**: JetBrains Mono, 12px, 400 weight (file paths, codec names)

### Components
- **Drop Zone**: Large rectangle, dashed amber border, hover effect
- **Format Buttons**: Rounded, grid layout, toggle on select
- **Progress Bar**: Amber fill on dark background
- **File List**: Table view with size, format, status columns

**Version**: 1.0
