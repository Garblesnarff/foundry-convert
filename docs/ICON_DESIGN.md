# App Icon Design Brief: FoundryConvert

## Concept
A stylized forge anvil with a cinematic film strip or media symbols being "hammered" into new shapes. The design should evoke transformation and craftsmanship.

## Color Palette
- **Background**: Deep charcoal (#141210) - Foundry primary
- **Anvil**: Dark metallic gray with subtle gradients
- **Sparks/Accents**: Forge amber (#E8A849) - glowing sparks flying from the anvil
- **Media symbols**: White or amber highlights

## Design Elements

### Central Anvil
- Classic blacksmith anvil shape
- Dark metallic shading
- Subtle amber glow at the horn (pointed end)

### Transformation Visual
- Film strip or media icon entering from left
- Multiple format symbols (play button, musical note, image icon) emerging from right
- Animated "spark" trails between input and output
- Sparks in forge amber (#E8A849)

### Icon States
- **Default**: Static anvil with sparks
- **Processing**: Animated sparks (for dock icon)
- **Error**: Red glow instead of amber

## Style Guidelines
- Flat design with subtle depth (neumorphism)
- High contrast for visibility at small sizes
- Clean vector shapes that scale well from 16x16 to 512x512
- Consistent with Foundry product line aesthetic

## Technical Notes
- Primary icon: 512x512 for App Store
- Generate all required sizes: 16, 32, 64, 128, 256, 512 (both 1x and 2x)
- Dark mode optimized (primary use case)
- Consider transparency for spark effects

## Reference
Similar to HandBrake's aesthetic but with more industrial/forge character and amber accents instead of green/orange.
